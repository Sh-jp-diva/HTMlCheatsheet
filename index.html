Your own FrontEnd Cheatsheet

HTML

HTML (HyperText Markup Language) is used to give content to a web page and instructs web browsers on how to structure that content.

Basic HTML Structure

<html>
<head>
<title> Website Title </title>
<body>
Content of website
</body>
</html>

Text Formatting

Tag Description
<h?>...</h?> Heading (?=number between 1 and 6, 1 is largest)
<b>...</b> Bold text
<i>...</i> Italic text
<u>...</u> Underline text
<strike>...</strike> Strikeout
<sup>...</sup> Superscript - Smaller text placed above normal text
<sub>...</sub> Subscript - Smaller text placed below normal text
<small>...</small> Small - Fineprint size text
<tt>...</tt> Typewriter text
<pre>...</pre> Pre-formatted text
<blockquote>...</blockquote> Quotation text block
<strong>...</strong> Strong - Shown as Bold in most browsers
<em>...</em> Emphasis - Shown as Italics in most browsers

Images

Tag Description
<img src="url" alt="text"> Basic image
<img> Tag attributes:
src="url" URL or filename of image (required!)
alt="text" Alternate Text (required!)(required!)
align="?" Image alignment within surrounding text (can use CSS instead)
